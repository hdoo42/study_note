``` python
plt.plot(x,y,"bo")
plt.plot(x,y,color= "b", marker = "o")
```

둘이 뭐가 다른거야?

두 코드는 **거의 동일한 결과**를 보여줍니다. 둘 다 $\mathbf{x}$와 $\mathbf{y}$ 데이터를 사용하여 **파란색**($\text{b}$) **원형**($\text{o}$) **마커**로만 이루어진 플롯을 생성합니다.

---

## 🎨 주요 차이점

두 표현식의 차이점은 스타일을 지정하는 **방식**에 있습니다.

### 1. 포맷 문자열 (`"bo"`) 사용

`plt.plot(x, y, "bo")`는 **포맷 문자열(Format String)**이라는 단축 문법을 사용합니다.

- `"b"`: 색상(Blue)
    
- `"o"`: 마커 모양(Circle)
    
- _참고: 선 스타일을 지정하지 않았기 때문에 선은 그려지지 않습니다._
    

이 방식은 코드가 짧고 간결하지만, 설정할 수 있는 옵션이 **제한적**입니다. 포맷 문자열은 **색상, 마커, 선 스타일**의 세 가지 속성만 순서에 관계없이 지정할 수 있습니다.

### 2. 키워드 인수 (`color`, `marker`) 사용

`plt.plot(x, y, color="b", marker="o")`는 **키워드 인수(Keyword Arguments)**를 직접 사용하여 속성을 지정합니다.

- `color="b"`: 색상 설정
    
- `marker="o"`: 마커 모양 설정
    

이 방식은 코드가 좀 더 길지만, **훨씬 더 많은 세부 설정**을 할 수 있습니다. 예를 들어, **마커 크기**(`markersize` 또는 `ms`), **마커 테두리 색상**(`markeredgecolor` 또는 `mec`), **알파 값**(`alpha`), **선 두께**(`linewidth` 또는 `lw`) 등 **수십 가지의 옵션**을 유연하게 제어할 수 있습니다.

---

## 🛠️ 언제 무엇을 사용해야 할까요?

- **간단한 플롯이나 빠른 시각화:** 포맷 문자열 (`"bo"`)을 사용하면 코드가 간결해집니다.
    
- **세부적인 커스터마이징이 필요할 때:** 키워드 인수 (`color="b", marker="o", markersize=10, linewidth=0`)를 사용하여 원하는 모든 속성을 설정하는 것이 좋습니다.
    

> **참고:** 두 번째 코드에서 마커만 표시하려면 **선 스타일을 명시적으로 설정하지 않거나** `linestyle='None'`을 추가해야 합니다. 하지만 `marker`만 지정하면 기본적으로 선은 그려지지 않습니다. 만약 **선과 마커를 동시에** 표시하고 싶다면, 포맷 문자열에는 선 스타일을 추가하고 (`"b-o"`), 키워드 인수에는 `linestyle="-"`와 `marker="o"`를 모두 사용하면 됩니다.
