IQR 공식은 다음과 같습니다.

$$\text{IQR} = \mathbf{Q_3} - \mathbf{Q_1}$$

따라서 빈칸을 채우면 다음과 같습니다.

$$\text{IQR} = \text{3사분위수 (Q₃)} - \text{1사분위수 (Q₁)}$$

---

### **용어 해설**

- **IQR (Interquartile Range, 사분위 범위):** 데이터를 크기 순으로 정렬했을 때, **3사분위수 (Q₃)** 와 **1사분위수 (Q₁)** 의 **차이**를 의미합니다. 이는 데이터의 중앙 50%가 얼마나 넓게 퍼져있는지를 나타내는 값으로, 데이터의 **산포도**를 측정하는 데 사용됩니다.
    
    - **$Q_3$ (3사분위수):** 전체 데이터를 크기 순으로 정렬했을 때, **상위 25%** 지점 (혹은 하위 75% 지점)에 해당하는 값입니다.
    
    - **$Q_1$ (1사분위수):** 전체 데이터를 크기 순으로 정렬했을 때, **하위 25%** 지점에 해당하는 값입니다.
        
- **아웃라이어 (Outlier, 이상치) 제거:**
    
    - IQR 방식은 이 IQR 값을 사용하여 데이터에서 **아웃라이어(이상치)** 를 정의하고 제거합니다.
        
    - 일반적으로 아웃라이어는 다음 범위 밖의 데이터 포인트로 정의됩니다.
        
        $$\text{[} Q_1 - 1.5 \times \text{IQR}, Q_3 + 1.5 \times \text{IQR} \text{]}$$
        
    - 즉, $Q_1$보다 $1.5 \times \text{IQR}$ 이상 작거나, $Q_3$보다 $1.5 \times \text{IQR}$ 이상 큰 값을 아웃라이어로 간주합니다.
