이해를 돕기 위해 **5개의 텍스트 데이터(과일 이름)** 를 예시로 들어, **정수 인코딩(Integer Encoding)**과 **원핫 인코딩(One-Hot Encoding)** 으로 변환되는 과정을 단계별로 보여드리겠습니다.

---

### **1. 예시 데이터 (일반 텍스트 배열)**

데이터의 개수는 5개이며, 종류(클래스)는 '사과', '바나나', '오렌지' 3가지로 가정합니다.

> **입력 데이터:** `['사과', '바나나', '사과', '오렌지', '바나나']`

---

### **2. 단계 1: 숫자로 변환 (Label Encoding)**

텍스트를 컴퓨터가 이해할 수 있도록 고유한 숫자를 부여합니다. (가나다순 혹은 등장 순서로 매핑)

- **매핑 규칙:**
    
    - 바나나 $\rightarrow$ 0
        
    - 사과 $\rightarrow$ 1
        
    - 오렌지 $\rightarrow$ 2
        

> **결과:** `[1, 0, 1, 2, 0]`

---

### **3. 단계 2: 원핫 인코딩 (One-Hot Encoding)**

위의 숫자들을 0과 1로만 이루어진 벡터로 변환합니다.

해당하는 숫자의 인덱스에는 1을, 나머지에는 0을 부여합니다.

**변환 규칙:**

- 0 (바나나) $\rightarrow$ `[1, 0, 0]`
    
- 1 (사과) $\rightarrow$ `[0, 1, 0]`
    
- 2 (오렌지) $\rightarrow$ `[0, 0, 1]`
    

**최종 결과 (행렬 형태):**

|**원래 데이터**|**숫자(Label)**|**원핫 인코딩 결과 (바나나, 사과, 오렌지 순)**|
|---|---|---|
|**사과**|1|`[0, 1, 0]`|
|**바나나**|0|`[1, 0, 0]`|
|**사과**|1|`[0, 1, 0]`|
|**오렌지**|2|`[0, 0, 1]`|
|**바나나**|0|`[1, 0, 0]`|

---

### **Python 코드로 구현하기 (Pandas 사용)**

실무에서 가장 많이 사용하는 파이썬 코드 예제입니다.

Python

```
import pandas as pd

# 1. 데이터 준비
data = ['사과', '바나나', '사과', '오렌지', '바나나']

# 2. 원핫 인코딩 (get_dummies 함수 사용)
one_hot_encoded = pd.get_dummies(data)

# 3. 결과 출력
print(one_hot_encoded)
```

**출력 결과:**

Plaintext

```
   바나나  사과  오렌지
0    0    1    0
1    1    0    0
2    0    1    0
3    0    0    1
4    1    0    0
```
