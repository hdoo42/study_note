L1 노름은 $||\mathbf{x}||_1$로 표기되며, 벡터의 모든 성분의 **절댓값을 합**하여 계산됩니다. 이는 **맨해튼 노름(Manhattan norm)** 또는 **택시 거리(Taxicab distance)** 라고도 불리는데, 이는 도시 격자(그리드)에서 두 지점 사이를 이동할 때 수평 또는 수직으로만 이동하여 측정하는 거리를 나타내기 때문입니다.

### 📝 공식

$\mathbf{x} = [x_1, x_2, \dots, x_n]$인 벡터에 대해, L1 노름은 다음과 같습니다:

$$||\mathbf{x}||_1 = \sum_{i=1}^{n} |x_i| = |x_1| + |x_2| + \dots + |x_n|$$

### 💡 예시

$\mathbf{x} = [3, -4, 2]$이면, 다음과 같습니다:

$$||\mathbf{x}||_1 = |3| + |-4| + |2| = 3 + 4 + 2 = 9$$

---

## 📐 L2 노름 (유클리드 노름)

L2 노름은 $||\mathbf{x}||_2$로 표기되며, 벡터 $\mathbf{x}$로 정의된 지점부터 원점까지의 표준적인 **유클리드 거리(Euclidean distance)** 를 나타내는 가장 일반적인 벡터 크기 측정값입니다.

### 📝 공식

L2 노름은 벡터 성분의 **제곱의 합에 대한 제곱근**으로 계산됩니다.

$$||\mathbf{x}||_2 = \sqrt{\sum_{i=1}^{n} x_i^2} = \sqrt{x_1^2 + x_2^2 + \dots + x_n^2}$$

### 💡 예시

$\mathbf{x} = [3, -4, 2]$이면, 다음과 같습니다:

$$||\mathbf{x}||_2 = \sqrt{3^2 + (-4)^2 + 2^2} = \sqrt{9 + 16 + 4} = \sqrt{29} \approx 5.385$$

## 🔑 주요 차이점

| 특징      | L1 노름($\|\|\mathbf{x}\|\|_1$)   | L2 노름($\|\|\mathbf{x}\|\|_2$) |
| ------- | ------------------------------- | ----------------------------- |
| 계산 방법   | 절댓값의 합                          | 제곱의 합에 대한 제곱근                 |
| 기하학적 의미 | 맨해튼/택시 거리                       | 유클리드(직선) 거리                   |
| 민감도     | 이상값(outliers)에 덜 민감함            | 큰 성분 값에 큰 패널티를 줌              |
| 사용 사례   | 특징 선택 (라쏘 회귀), 희소성(Sparsity) 유도 | 일반적인 크기, 표준 거리, 평균 제곱 오차(MSE) |


L1 노름과 L2 노름이 **머신러닝**에서 정규화(regularization)를 위해 어떻게 사용되는지 알고 싶으신가요?
