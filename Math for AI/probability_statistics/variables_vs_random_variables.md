# Variables vs Random Variables

## General Concept

### 일반 변수
- 확정적인 값을 가집니다.
- 수학적 방정식이나 알고리즘에서 특정 값을 저장하는 데 사용됩니다.
- 값이 정해지면 변하지 않습니다(상수화됨).

### 확률 변수
- 무작위 현상의 결과에 따라 값이 결정됩니다.
- 특정 확률 분포를 따릅니다.
- 실현값(realization)이 사전에 정확히 예측될 수 없습니다.

## 머신러닝/인공지능에서의 차이점

모든 변수가 확률 변수는 아닙니다. 확률 변수는 우리가 관심을 갖는 대상, 즉 예측하거나 모델링하려는 대상입니다:

### 지도학습
예측하려는 $y$ 값이 확률 변수가 됩니다. 입력 $x$는 일반적으로 고정된 값으로 처리됩니다.
- 회귀 문제에서 목표 변수 $y$는 연속 확률 변수로 모델링됩니다.
- 분류 문제에서 클래스 레이블 $y$는 이산 확률 변수로 모델링됩니다.

### 비지도학습
관측된 데이터 $x$ 자체가 확률 변수로 간주됩니다.
- 클러스터링에서 데이터 포인트의 분포를 모델링합니다.
- 생성 모델에서 입력 데이터는 특정 확률 분포에서 샘플링된 것으로 가정합니다.

### 강화학습
상태 전이와 보상이 확률 변수로 모델링됩니다.

## 예시

### 선형 회귀
선형 회귀에서:
- $y = wx + b + \epsilon$ 
- $y$: 확률 변수 (예측하려는 타겟)
- $x$: 일반 변수 (입력 데이터)
- $w, b$: 일반 변수 (모델 파라미터)
- $\epsilon$: 확률 변수 (노이즈)

### 딥러닝
- 신경망 가중치($W$)와 편향($b$)은 일반 변수입니다.
- 베이지안 신경망에서는 가중치가 확률 변수로 취급될 수 있습니다.
- 드롭아웃은 특정 뉴런을 확률적으로 꺼지게 만들어, 확률 변수처럼 동작하게 합니다.

## 결론

본질적으로, 우리가 예측하고자 하는 불확실한 대상이 확률 변수가 됩니다. 이 불확실성을 확률 분포로 모델링함으로써 예측과 의사결정에 수학적 프레임워크를 적용할 수 있습니다. 변수라고 다 확률 변수가 아니며, 결국 우리가 보고싶은 것, 관심있는 대상이 확률 변수가 됩니다.
