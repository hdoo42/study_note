# 기본 주행 방법

이 문서는 처음 운전을 배우는 초보자를 위한 기본 주행 방법에 대한 내용입니다.

## 출발하기

1. **브레이크 페달 밟기**
   - 출발 전 항상 브레이크 페달을 밟은 상태에서 시작

2. **기어 변속**
   - P에서 D로 변속 (전진 시)
   - P에서 R로 변속 (후진 시)

3. **주차 브레이크 해제**
   - 전자식: 버튼 조작
   - 수동식: 레버 내리기

4. **주변 안전 확인**
   - 미러를 통한 주변 확인
   - 맹점(사각지대) 확인을 위해 고개 돌려 확인

5. **신호 표시**
   - 방향지시등 작동 (필요시)

6. **부드러운 출발**
   - 브레이크 페달을 서서히 놓음
   - 차량이 천천히 움직이기 시작하면 가속 페달을 부드럽게 밟음

## 기본 주행 조작

1. **가속하기**
   - 가속 페달을 부드럽게 밟아 속도 증가
   - 급가속은 피하기

2. **감속하기**
   - 가속 페달에서 발을 떼고 자연스럽게 감속
   - 필요시 브레이크 페달을 부드럽게 밟아 감속

3. **정지하기**
   - 브레이크 페달을 점진적으로 밟아 부드럽게 정지
   - 급정거는 피하기

4. **핸들 조작**
   - 양손으로 핸들 잡기 (9시와 3시 방향 권장)
   - 부드럽고 일정한 속도로 핸들 돌리기

## 코너링 (회전하기)

1. **회전 전 준비**
   - 미리 속도 줄이기
   - 방향지시등 켜기 (우회전은 오른쪽, 좌회전은 왼쪽)

2. **회전 중**
   - 적절한 속도 유지
   - 부드럽게 핸들 조작

3. **회전 후**
   - 차선 중앙에 맞추어 주행
   - 방향지시등 자동 복귀 확인

## 차선 변경

1. **차선 변경 전 확인**
   - 미러로 후방 확인
   - 맹점(사각지대) 확인을 위해 고개 돌려 확인

2. **방향지시등 작동**
   - 변경하려는 방향의 지시등 켜기

3. **차선 변경 실행**
   - 부드럽게 핸들을 돌려 차선 변경
   - 급격한 조작 피하기

4. **차선 변경 후**
   - 정상 주행 자세로 복귀
   - 방향지시등 자동 복귀 확인

## 안전 거리 유지

1. **앞차와의 거리**
   - 최소 2초 이상의 거리 유지 (2초 룰)
   - 악천후 시에는 더 긴 거리 유지

2. **속도에 따른 안전 거리**
   - 속도가 높을수록 더 많은 안전 거리 필요

## 주차하기

1. **주차 공간 확인**
   - 충분한 크기의 주차 공간 확인
   - 주차 제한 표시 확인

2. **주차 실행**
   - 속도 줄이기
   - 방향지시등 켜기
   - 적절한 주차 방법 선택 (평행, 직각 등)

3. **주차 완료**
   - 기어를 P로 변경
   - 주차 브레이크 작동
   - 엔진 정지

자세한 주차 방법은 [[drive/parking_methods]] 문서를 참조하세요.

출발 전 기본 조작방법은 [[drive/pre_driving_operations]] 문서에서 확인할 수 있습니다.
