# 출발 전 기본 조작방법

이 문서는 처음 운전을 배우는 초보자를 위한 출발 전 기본 조작방법에 대한 내용입니다.

## 운전 시작 전 체크 사항

1. **운전석 조정**
   - 좌석 위치 조정: 페달을 끝까지 밟을 수 있도록 적절한 거리 확보
   - 좌석 높이 조정: 전방 시야가 확보되는 높이로 조절
   - 등받이 각도 조정: 약간 뒤로 기울어진 편안한 자세로 조절
   - 헤드레스트 조정: 머리 중앙에 오도록 높이 조절

2. **미러 조정**
   - 룸미러: 뒷유리 전체가 보이도록 조정
   - 사이드미러: 차량 측면이 약간 보이고 후방이 잘 보이도록 조정

3. **안전벨트 착용**
   - 안전벨트를 꼬이지 않게 매고, 골반과 가슴을 가로지르도록 조정

4. **스티어링 휠(핸들) 조정**
   - 높이와 거리를 조절하여 편안하게 잡을 수 있도록 조정

## 기본 조작 장치 이해하기

1. **기어 변속 장치**
   - P(주차): 주차 시 사용
   - R(후진): 차량 후진 시 사용
   - N(중립): 엔진 동력이 전달되지 않는 상태
   - D(주행): 일반 주행 시 사용

2. **페달**
   - 우측: 가속 페달(액셀)
   - 중앙: 브레이크 페달
   - 좌측: 클러치 페달(수동변속기 차량만 해당)

3. **사이드 브레이크(주차 브레이크)**
   - 주차 시 차량 고정을 위해 사용
   - 출발 전 해제 필수

4. **방향지시등(윙커)**
   - 방향 전환 시 사용
   - 레버를 위로: 우회전 신호
   - 레버를 아래로: 좌회전 신호

5. **라이트 조작**
   - 전조등, 차폭등, 비상등 조작법 이해
   - 상향등과 하향등 전환 방법

6. **와이퍼 조작**
   - 와이퍼 속도 조절 방법
   - 워셔액 분사 방법

## 출발 전 차량 외부 점검

1. **타이어 상태 확인**
   - 공기압 및 마모 상태 확인

2. **각종 등화장치 확인**
   - 전조등, 브레이크등, 방향지시등 작동 여부

3. **차량 주변 안전 확인**
   - 차량 하부 및 주변 장애물 확인

## 시동 걸기

1. **시동 걸기 전 확인사항**
   - 기어 P위치 확인
   - 주차 브레이크 작동 확인
   - 안전벨트 착용 확인

2. **시동 거는 방법**
   - 브레이크 페달 밟기
   - 키 또는 시동 버튼으로 시동 걸기
   - 엔진 소리와 계기판 확인

3. **시동 후 확인사항**
   - 각종 경고등 확인
   - 연료량 확인
   - 엔진 상태 확인

## 출발 준비

1. **기어 변속**
   - 브레이크 페달을 밟은 상태에서 기어를 D(주행) 또는 R(후진)로 변경

2. **주차 브레이크 해제**
   - 주차 브레이크 레버를 내리거나 버튼을 눌러 해제

3. **주변 안전 확인**
   - 미러와 시야를 통해 차량 주변 확인
   - 사각지대 확인을 위해 고개 돌려 확인
