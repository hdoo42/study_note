# 주차 방법

이 문서는 처음 운전을 배우는 초보자를 위한 다양한 주차 방법에 대한 내용입니다.

## 주차 전 준비사항

1. **주차 공간 확인**
   - 차량 크기에 적합한 공간인지 확인
   - 주차 제한 표시 확인 (주차 금지, 시간제한 등)

2. **주차 방법 결정**
   - 상황에 맞는 주차 방법 선택 (평행, 직각, 사선 등)

3. **주변 안전 확인**
   - 보행자, 다른 차량 확인
   - 지면 상태 및 장애물 확인

## 직각 주차 (앞으로 진입)

1. **위치 선정**
   - 주차공간 앞에 충분한 거리 확보

2. **주차 실행**
   - 방향지시등 켜기
   - 속도 줄이기
   - 주차공간 중앙을 향해 천천히 진입
   - 차량이 공간 중앙에 위치하도록 조정

3. **마무리**
   - 브레이크 밟기
   - 기어 P로 변경
   - 주차 브레이크 작동

## 직각 주차 (후진 진입)

1. **위치 선정**
   - 주차공간을 지나 약 1~2m 앞에 정차
   - 차량을 주차공간과 평행하게 위치

2. **후진 준비**
   - 방향지시등 켜기
   - 주변 안전 확인
   - 기어를 R로 변경

3. **후진 실행**
   - 천천히 후진하며 핸들 돌리기
   - 사이드미러와 후방카메라 활용
   - 차량이 공간 중앙에 위치하도록 조정

4. **마무리**
   - 브레이크 밟기
   - 기어 P로 변경
   - 주차 브레이크 작동

## 평행 주차 (길가 주차)

1. **위치 선정**
   - 주차할 공간 옆의 차량과 나란히 정차
   - 앞 차량과 약 1m 간격 유지

2. **후진 준비**
   - 방향지시등 켜기
   - 주변 안전 확인
   - 기어를 R로 변경

3. **진입 시작**
   - 천천히 후진
   - 차량 뒷부분이 주차공간에 들어설 때 핸들을 주차공간 쪽으로 최대한 돌림

4. **차량 위치 조정**
   - 옆 차량과 나란해지기 시작하면 핸들을 반대로 돌림
   - 차량이 주차공간에 평행하게 위치하도록 조정

5. **마무리**
   - 필요시 전진/후진하며 위치 미세 조정
   - 기어 P로 변경
   - 주차 브레이크 작동

## 사선 주차

1. **위치 선정**
   - 주차공간 앞에 적당한 거리 확보

2. **주차 실행**
   - 방향지시등 켜기
   - 사선 방향에 맞추어 진입
   - 천천히 진행하며 주차선 안에 차량이 위치하도록 조정

3. **마무리**
   - 브레이크 밟기
   - 기어 P로 변경
   - 주차 브레이크 작동

## 주차 완료 후 확인사항

1. **차량 위치 확인**
   - 주차선 내에 올바르게 주차되었는지 확인
   - 다른 차량의 출입에 방해되지 않는지 확인

2. **차량 보안**
   - 모든 창문 닫기
   - 도어락 확인
   - 귀중품 보이지 않게 보관

3. **출차 경로 확인**
   - 나중에 출차할 때의 경로 확인
   - 일방통행, 출구 위치 등 확인

기본 주행 방법은 [[drive/basic_driving]] 문서에서 확인할 수 있습니다.
